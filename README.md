# Quiz App

A simple quiz application with gamification features.

## Setup Instructions

Follow these steps to set up and run the project on your local machine:

1. **Clone the repository:**
   ```sh
   git clone https://github.com/VishwasGandham/quiz-app.git
   ```

cd quiz-app

npm install

npm run dev
